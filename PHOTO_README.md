# Add Your Photo Here

To add your photo to the resume:

1. **Prepare your photo:**
   - Use a professional headshot
   - Square or portrait orientation works best
   - Recommended size: at least 400x400 pixels
   - File format: JPG, PNG, or WebP

2. **Save the photo:**
   - Name it `profile-photo.jpg` (or update the HTML if using a different name)
   - Place it in the same directory as your `index.html` file

3. **Photo guidelines:**
   - Professional appearance
   - Good lighting and background
   - Recent photo
   - Neutral expression

The photo will appear as a circular image next to your name and contact information on desktop, and below on mobile devices.
